# Week2_Project
Titanic Dataset Analysis Using Pandas
Project Overview

This project analyzes the Titanic passenger dataset using Python’s Pandas library. It focuses on applying core Pandas concepts such as Series, DataFrames, data cleaning, filtering, grouping, merging, and aggregation to understand passenger survival patterns.

Project Objective

Analyze survival trends among Titanic passengers

Apply fundamental Pandas operations on a real-world dataset

Identify key factors affecting survival such as age, gender, class, and fare

Dataset Description

The dataset contains the following columns:

PassengerId

Name

Sex

Age

Pclass

Fare

Survived

Embarked

SibSp

Parch

Dataset Source: Public Titanic dataset from Kaggle

Tools and Technologies

Python

Pandas

Jupyter Notebook / Python IDE

Concepts Used

CSV file import using Pandas

Series and DataFrame creation

Data inspection and summary statistics

Handling missing values

Indexing and slicing

Filtering and sorting data

Grouping and aggregation using groupby()

Pivot tables

Merging and concatenation

Basic statistical calculations

Key Analysis Performed

Survival rate by gender

Average age by passenger class

Survival count by embarkation location

Fare-based survival comparison

Survivor vs non-survivor age analysis

Key Insights

Female passengers had a higher survival rate than males

First-class passengers showed better survival outcomes

Higher fares were associated with higher survival chances

Younger passengers had improved survival probability

Socio-economic status played a major role in survival

Conclusion

This project demonstrates the effective use of Pandas for data analysis and provides insights into survival patterns from the Titanic dataset. It serves as a foundational project for understanding data manipulation and analysis using Python.
